# Amerre

"Welcome to Amerre" is an abstract narration based platform game in development.
This version features the first of four planned chapters.
The game requires at least an GPU with OpenGL 3.2 support to run.

This version of the game has no in-game settings implemented yet.
The game is preconfigured to run decent on most hardware.
If you have issues run

Amerre.exe --help
on Windows

or
./amerre --help
on Linux too see the supported options
